# Materials for PDAI26

## Contents: 

- 2_streamlit: Scripts for the `streamlit` demos.

## Cloning the repository

```bash
git clone https://github.com/prof-jose/PDAI26_
```

## Packages
To install the required packages, `venv` is recommended. It is assumed you know how to create and activate a virtual environment. Install the packages using the following command:
```bash
pip install -r requirements.txt
```
